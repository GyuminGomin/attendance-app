## 프로젝트 상세
```
일정관리앱

1. 일정을 기록한다.
2. 앱은 컴퓨터가 켜지자마자 실행프로그램으로 등록되어 있어야 한다.
3. 앱이 실행되면, AI가 그 일정에 맞춰 UTF8 시간에 맞춰 일정알림을 해준다.
    - 앱이 실행될 때, 자동으로 오늘날짜 출석을 완료
    - 앱이 실행될 때, 오늘날짜 이미 체크되어 있으면 이미 체크되어 있다고 표시
        - 생각해보니 앱이 실행될 때, 자동 출석체크보다는 일정을 작성 후 완료했을 경우 출석체크 하는 걸로 변경
    - 날짜 클릭 시 markdown으로도 볼 수 있게 설정

4. 일정이 제시각에 완료되면, 일정에 체크여부가 판단되고 자동으로 체크가 된다.
    - 이 조건을 개발하면서 사용해보면서 바꿔가자
5. 추가로 일정에 잘 맞춰지면 코인제도를 통해 가상코인화를 해보자
```

## 해야할 내용 정리
```
- 저장 버튼 클릭 시 오늘 체크 o
- v-chip 다른 날짜 선택했을 시 뜨지 않게 변경 o
- 오늘 기준으로는 체크 표시가 잘 뜨는데, 내일도 체크된 상태면 잘 뜨는지 확인
    - 오늘날짜만 색깔이 표시가 되도록 설정되어 있음
        - attendance.json 파일에서 날짜기준으로 한달을 돌려 checked:true 면 출석했다(저장했다)라는 표시가 보이도록 설정
        - 수정은 오늘날짜만 가능
        - 하지만 다른 날짜 클릭 시 포커스가 바뀌는 느낌 주기

- 수정하고 있을 때 앱을 끄지 않으면 임시데이터 저장상태로 유지 필요
- markdown 뷰로 볼 수 있게 설정


- 강의를 듣기 전에 일단 안풀려서 짜증이 나더라도 오랜시간 생각을 거듭해서 문제 해결을 다 하고 이 앱이 완료되면, vue 강의를 추가로 더 듣자 (이러는 게 더 메리트있을 것 같다는 생각)
(Why ? - 현재 제작하고 있는 앱은 내 삶에 도움이 됨과 동시에 어렵지 않은 앱이기 때문)
```
