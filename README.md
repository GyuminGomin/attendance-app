## 프로젝트 상세
```
1. 일정을 기록한다.

2. 앱은 컴퓨터가 켜지자마자 실행프로그램으로 등록되어 있어야 한다.

3. 앱이 실행되면, 오늘 일정을 기록해야 한다.

  - 수정페이지에서 오늘날짜만 수정 가능

  - 다른날짜는 수행도와 확인만 가능

  - 이미 작성된 것은 markdown으로 작성된 것도 미리보기로 확인 가능

4. 저장이 되면, 오늘 일정을 분석 후 시간에 따라 해야할 일정을 알림해준다.

5. 가능하다면, 해야할 일정을 리스트화 해 수행했다고 판단이 되면 진행도를 %로 나타내준다.

6. 진행도에 따라 코인제도를 적용해 코인을 어느정도 지급해준다.
```

## 해야할 내용 정리
```
1. 기존 앱 시작시 바로 출석체크 -> 저장 시 출석체크

2. 다른날짜 클릭 시 최상단에 뜨는 저장완료 Chip 생략

3. 기존 오늘날짜만 색상이 표시되는 부분을 출석이 완료되었을 때, 포커스가 이동되었을 때 느낌이 살 수 있도록 수정

4. Ctrl + S 단축키 클릭 시 자동 저장

5. 디자인 : 크기 (가로 최소 650px)

6. 수정하고 있을 때 앱을 끄지 않으면 임시데이터 저장상태 유지 필요
    - 다른 날짜는 수정이 불가능하므로 임시데이터 필요가 없음
    - 오늘 날짜에서 갑자기 다른 날짜가 보고 싶을 때, 임시데이터 유지 장치

7. markdown 뷰로 볼 수 있게 설정


- 강의를 듣기 전에 일단 안풀려서 짜증이 나더라도 오랜시간 생각을 거듭해서 문제 해결을 다 하고 이 앱이 완료되면, vue 강의를 추가로 더 듣자 (이러는 게 더 메리트있을 것 같다는 생각)
(Why ? - 현재 제작하고 있는 앱은 내 삶에 도움이 됨과 동시에 어렵지 않은 앱이기 때문)
```
